React Window Decorators

This is demo for react-window-decorators. Library includes two decorators (higher order components):

  • withScroll - tracks window scroll and injects scrollPositionX and scrollPositionY prop to your component.
  • withWindow - tracks window dimensions (width/height), orientation, breakpoint* and isTouchDevice and injects them all as props to your component.
    • * breakpoint configuration must be passed to it

More detailed readme and documentation are available on GitHub.
